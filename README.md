# Formation React

Formation React pour l'agence numerique

## Informations pratiques

* Pré-requis : ``node.js``
* Installation des dépendances : ``npm install``
* Constructions des slides : ``npm run build`` (dans le dossier ``build``)
* Serveur live-reload : ``npm run serve``
